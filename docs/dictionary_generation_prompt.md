# Prompt для генерации словарного пака

Сгенерируй словарный пак для веб-приложения "Тренажер английского словаря".

Требования:
- язык слов: английский;
- переводы: на русский язык;
- количество слов: 75;
- уровни CEFR: A1, A2, B1, B2, C1, по 15 слов на каждый уровень;
- каждое английское слово должно быть в канонической форме: единственное число для существительных, базовая форма для глаголов, без дублей;
- не добавляй множественное число, если есть базовая форма: book вместо books, study вместо studies, child вместо children;
- не используй слишком редкие или устаревшие слова;
- темы: повседневная речь, учеба, работа, технологии, академические навыки;
- для каждого слова укажи транскрипцию, часть речи и пример предложения.

Верни результат строго в JSON-массиве без пояснений. Формат одного элемента:

```json
{
  "english": "example",
  "translation": "пример",
  "transcription": "[ɪɡˈzɑːmpəl]",
  "example": "This is a clear example.",
  "level": "A2",
  "part_of_speech": "noun",
  "category": "Study and work"
}
```

Разрешенные значения `part_of_speech`: `noun`, `verb`, `adjective`, `adverb`, `phrase`, `other`.

Поле `category` используется как служебная тематическая группа для админки. На сайте основной фильтр должен оставаться по уровням.

Разрешенные тематические группы: `Core vocabulary`, `Communication`, `Study and work`, `Technology`, `Academic skills`.
